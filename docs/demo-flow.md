# 项目演示流程

## 准备

1. 执行 `database/schema.sql` 创建数据库和表。
2. 执行 `database/init.sql` 初始化角色、测试用户、题目和测试试卷。
3. 启动 MySQL 和 Redis。
4. 修改 `src/main/resources/application.yml` 中的数据库用户名和密码。
5. 启动 Spring Boot 项目。
6. 浏览器访问 `http://localhost:8080/login.html`。

## 演示账号

- 管理员：admin / 123456
- 教师：teacher01 / 123456
- 学生：student01 / 123456

## 主流程

1. 使用教师账号登录。
2. 进入题库管理，新增一个数据库原理题目。
3. 进入试卷管理，创建一张新试卷。
4. 进入试卷组题页面，从题库选择题目加入试卷。
5. 发布试卷，系统自动汇总总分。
6. 退出登录，使用学生账号登录。
7. 进入参加考试，点击开始考试。
8. 在线答题，提交试卷。
9. 查看我的成绩，打开考试详情。
10. 查看错题本。
11. 退出登录，使用教师账号查看成绩统计。
12. 使用管理员账号登录，查看用户列表和登录日志。

## 答辩重点说明

1. 题目来自 `question` 表，不写死在代码中。
2. 试卷和题目通过 `paper_question` 表建立多对多关系。
3. 获取考试题目接口不返回正确答案。
4. 提交试卷在后端判分，并使用事务保证一致性。
5. 错题保存到 `wrong_question`，用于学生错题本。
6. 成绩查询使用 `v_student_score` 视图。
7. 登录使用 BCrypt、JWT 和 Redis，支持 Token 过期和退出登录。
